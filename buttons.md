# Buttons
We love 'em! Press them, switch them, click them, they can do anything! And Arbiter has buttons in it too!

Buttons are interactible in-world displays mixed with interactions that let players control different things while being under the same ticking process. Amount of buttons, their position, behavior, etc. is dictated by the button type.

**Check out `arbiterlib:button/example/`** function folder for more!

Run `/function arbiterlib:button/example/2_around_text/spawn` in game to check out buttons for yourself!
